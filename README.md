# Food Delivery Funnel Analysis

Analyzing user behavior across a food ordering funnel to identify drop-offs and improve conversion.

---

## 🧩 Problem

In food delivery platforms, users drop off at various stages of the ordering process. Identifying where and why this happens is critical to improving conversion rates.

---

## 📊 Dataset

- Simulated dataset of ~10,000 users  
- Funnel stages:
  - App Open
  - Restaurant View
  - Menu View
  - Add to Cart
  - Order Placed  

---

## 🔍 Analysis

- Calculated conversion rates at each stage  
- Identified major drop-off points  
- Analyzed user behavior patterns  

---

## 📉 Key Insight

- ~50% drop-off at the **menu decision stage**  
- Indicates friction in:
  - decision making  
  - menu clarity  
  - overload of options  

---

## 💡 Solution / Hypothesis

Proposed improvements:

- Simplified menu UI  
- Highlight popular items  
- Add “recommended for you”  
- Reduce cognitive load  

---

## 🧪 Experiment Design

A/B Test:

- Control: existing menu  
- Variant: simplified + recommendations  

**Expected result:**
- Add-to-cart conversion improves from 50% → 65%  

---

## 📈 Impact

- Improved conversion rate  
- Better user experience  
- Increased order volume  

---

- SQL (joins, aggregations)
